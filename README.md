# 🤖 AI FAQ Assistant

An intelligent FAQ chatbot built using **Python**, **Pandas**, and **Streamlit**. The application answers user questions using **Exact Matching**, **Fuzzy Search**, and **Keyword Matching** through a modern interactive chat interface.

## 📖 Project Overview

The **AI FAQ Assistant** is a web-based chatbot designed to answer frequently asked questions from a predefined knowledge base.

The chatbot intelligently understands user queries using:

* ✅ Exact Question Matching
* ✅ Fuzzy Search (Handles Typing Mistakes)
* ✅ Keyword-Based Search
* ✅ Greetings & Small Talk
* ✅ Interactive Chat Interface

This project was developed as part of the **Horizon TechX Internship** to demonstrate Python programming, chatbot development, and Streamlit web application skills.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Balaji-Analytics/AI-FAQ-Chatbot.git
```

Move into the project directory

```bash
cd AI-FAQ-Chatbot
```

Install the required packages

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

Open your browser and visit

```
http://localhost:8501
```

## 🧠 Search Methods

### ✅ Exact Match

Returns an answer when the user's question exactly matches an FAQ.

### ✅ Fuzzy Match

Uses Python's Difflib library to recognize similar questions, even with spelling mistakes.

Example:

```
What is Pyhton?
```

↓

```
What is Python?
```

### ✅ Keyword Match

If no exact or fuzzy match is found, the chatbot searches for the most relevant FAQ based on keywords.
